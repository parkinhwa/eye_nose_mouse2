# 눈코입 맞추기 게임

Phaser 3로 제작한 재미있는 얼굴 파츠 맞추기 게임입니다.

## 게임 방법

1. 시작 화면에서 '시작하기' 버튼을 클릭하거나 스페이스바를 누릅니다
2. 왼쪽 눈, 오른쪽 눈, 코, 입 순서로 파츠가 위에서 아래로 떨어집니다
3. 각 파츠가 정확한 위치에 왔을 때 **스페이스바**를 눌러 고정합니다
4. 정확도에 따라 점수가 부여됩니다 (최대 100점/파츠)
5. 모든 파츠를 배치한 후 최종 점수를 확인합니다

## 점수 시스템

- **90점 이상**: 완벽! (S급)
- **80-89점**: 좋아요! (A급)
- **70-79점**: 괜찮아요 (B급)
- **60-69점**: 조금 더! (C급)
- **60점 미만**: 아쉬워요 (D급)

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 게임 실행
npm start
```

브라우저가 자동으로 열리며 게임이 시작됩니다 (기본 포트: 8080)

## 기술 스택

- Phaser 3.80.1
- HTML5 Canvas
- JavaScript (ES6)

## 조작법

- **스페이스바**: 떨어지는 파츠 고정 / 게임 재시작
- **마우스 클릭**: 버튼 조작
- **ESC**: 처음 화면으로 (결과 화면에서)
